# tibiz
jdk 1.8
springboot
sts4


기동방법
resource 폴더 application.yml 파일에서
profiles:
	active: local 확인 후 
springboot로 실행
시작 실패시 프로젝트 우측 클릭 후
maven -> update project 클릭하고 재시작

현재 계정은 qnfdydtktk/test 로 로그인 가능하며 다중접속 불가능
localhost:8880/admin/mainView.do
해당 url이 관리자 관련 기능들이며

현재 회원가입, 패스워드찾기 등 일부 외부api를 사용하는 기능들 사용 불가능
